## 🔥 Fitur Lengkap (v1.2.0)

### 1. Universal Website Detection *(NEW!)*
Bekerja di **semua jenis website**, bukan hanya WordPress:
- **Technology Detector**: Mendeteksi teknologi yang digunakan (Next.js, React, Nuxt.js, Angular, Laravel, Shopify, Blogger, Joomla, dll).
- **WordPress Mode**: Jika situs WordPress terdeteksi, secara otomatis masuk ke mode penuh dengan fitur tema, plugin, dan stealth detection.
- **Auto Script Injection**: Ekstensi kini secara otomatis menyuntikkan skrip analisis ke tab yang sudah terbuka tanpa perlu refresh halaman terlebih dahulu.
- **System Page Protection**: Halaman internal browser (`chrome://`, `edge://`, dll) ditangani dengan pesan yang jelas — tidak ada lagi loading yang stuck.

### 2. Deep Core Detection *(WordPress)*
Mendeteksi jeroan website WordPress yang sering disembunyikan:
- **Theme Detector**: Mengetahui nama tema dan versinya.
- **Plugin Scanner**: Mendeteksi daftar plugin yang aktif.
- **Stealth Mode Bypass**: Tembus deteksi plugin keamanan seperti "Hide My WP Ghost" atau struktur folder custom.
- **Schema & Structured Data Detector**: Otomatis mendeteksi tipe structured data yang terpasang di halaman (JSON-LD & Microdata) secara visual.

### 3. Google Search Console & Bing Webmaster Tools
- **One-Click Indexing GSC**: Request index URL aktif secara instan ke GSC URL Inspection.
- **One-Click Submit Bing**: Kirimkan URL aktif ke Bing Webmaster Tools dengan satu klik.
- **Instant GSC Report**: Buka Laporan Performa pencarian di Google Search Console langsung untuk periode 3, 6, 12, atau 16 bulan terakhir.
- **Instant URL Index Checker**: Cek apakah URL tertentu sudah terindeks di Google dan Bing dengan jalan pintas query `site:`.

### 4. Server & Security Intelligence
- **Hosting & IP**: Mengungkap Provider Hosting, Alamat IP, dan Lokasi Server (Negara/Kota).
- **Nameserver (NS)**: Melihat DNS Nameserver yang digunakan.
- **CDN Detector**: Mendeteksi penggunaan CDN/Proxy (Cloudflare, Sucuri, Akamai, dll).
- **Security Check**: Validasi status HTTPS di semua website.

### 5. SEO & Content Tools (Powered by AI)
- **AI Article Rewriter**: Tulis ulang konten artikel website tersebut menjadi unik dan SEO-Friendly dalam satu klik — kini bekerja di semua jenis website.
- **Keyword Intelligence**: Ekstrak keyword dari konten + Saran LSI Keyword dari AI.
- **AI Traffic Potential**: Estimasi potensi traffic keyword (High/Medium/Low) dengan indikator warna.
- **External Traffic Insights**: Shortcut ke SEMrush, Ahrefs, dan Ubersuggest.

### 6. Smart Audit System (Universal)
Audit kesehatan website secara instan dengan skor indikator — bekerja di semua jenis website:
- **SEO Health**: Cek Meta Title, Description, H1, dan Open Graph.
- **Performance**: Hitung jumlah script, style, dan aset gambar.
- **Accessibility**: Cek kelengkapan Alt text gambar & HTML Language attribute.
- **Mobile Readiness**: Cek viewport meta tag.
- **AI Suggestions**: Dapatkan saran perbaikan teknis otomatis dari AI berdasarkan data audit.

### 7. Developer Experience & UI
- **Modern Tab Navigation**: Akses rapi ke panel Overview, SEO Tools, GSC & Bing, dan Server.
- **Wider Design**: Ukuran popup 480px agar pembacaan data lebih nyaman.
- **Smart Cache**: Hasil analisis AI disimpan per tab (hemat kuota API dan loading instan saat dibuka kembali).
- **Glassmorphism UI**: Tampilan modern, responsif, dan nyaman di mata.
- **Privacy First**: API Key Anda tersimpan lokal di browser masing-masing.

## 📦 Instalasi & Penggunaan

1. **Download Source Code**: Clone atau download repository ini.
2. **Chrome Extensions**: Buka `chrome://extensions/` di browser Anda.
3. **Developer Mode**: Aktifkan toggle di pojok kanan atas.
4. **Load Unpacked**: Klik tombol ini dan pilih folder `wpaudet`.
5. **Setup AI**: Klik icon gear (⚙️) di dalam extension untuk memasukkan API Key (OpenAI/Gemini/Groq/DeepSeek) agar fitur AI berjalan maksimal.

---

## 📋 Catatan Penting

- **WHOIS**: Fitur WHOIS mendukung domain `.com`, `.net`, `.org`, dan `.id`. Domain dengan TLD lain (`.io`, `.co`, `.xyz`, dll) tidak dapat dicek langsung karena keterbatasan CORS pada Manifest V3. Akan ditampilkan link untuk cek manual.
- **Halaman Sistem Browser**: Ekstensi tidak bisa menganalisis halaman internal seperti `chrome://extensions/` atau `about:blank`.
- **Auto Injection**: Jika Anda baru saja menginstal atau mereload ekstensi, cukup buka tab yang ingin dianalisis — tidak perlu refresh halaman tersebut (akan otomatis diinjeksi).
