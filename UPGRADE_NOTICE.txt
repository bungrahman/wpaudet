**IMPORTANT FOR EXISTING USERS:**
Because this update adds new server permissions (RDAP/WHOIS servers) to `manifest.json`, Chrome will NOT auto-update the extension code in memory.
**YOU MUST RELOAD THE EXTENSION MANUALLY:**
1. Go to `chrome://extensions/`
2. Find WPAudet.
3. Click the Reload (🔄) button.

If you don't do this, WHOIS features might show errors.
